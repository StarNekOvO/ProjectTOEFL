# 错题记录目录

这个目录用于存放所有的错题记录。

## 目录结构
- `reading/` - 阅读错题
- `listening/` - 听力错题
- `speaking/` - 口语问题
- `writing/` - 写作问题
- `vocabulary/` - 词汇问题

## 文件命名建议
为了便于查找和管理，建议使用以下命名格式：

- **TPO题目**：`tpo45_reading_q12.md`（TPO45阅读第12题）
- **改革后真题**：`reform_2024_01_listening_q5.md`（2024年1月改革后真题听力第5题）
- **听力专项**：`listening_special_q8.md`（听力专项第8题）
- **词汇专项**：`vocab_word_abandon.md`（词汇专项 - abandon）

## 如何使用
1. 复制 `templates/error_template.md` 到对应的子目录
2. 重命名为有意义的文件名
3. 填写错题信息
4. 在"需要AI分析的问题"部分列出你的问题
5. 告诉我："请帮我分析这个错题"

## 快速查找
- 按日期查找：文件名包含日期
- 按来源查找：文件名包含tpo/reform/listening/vocab
- 按题型查找：在对应的子目录中查找
