# ProjectTOEFL - 托福学习计划与错题分析系统

## 📋 项目简介

这是一个专为ADHD学习者设计的托福备考项目，帮助你：
- 📅 制定适合2小时集中注意力的学习计划
- 📝 记录和整理错题
- 🤖 获得AI辅助的错题分析
- 📊 追踪学习进度

## 🎯 学习计划

### 时间安排原则
- **周末集中学习**：2小时完成1套改革后真题（完整模考）
- **平日碎片学习**：30-60分钟完成TPO单项练习（听力/阅读/口语/写作/词汇）

### 学习材料
1. **TPO75套** - 用于平日单项练习
2. **2023改革后托福真题100+套** - 用于周末完整模考
3. **听力专项** - 平日听力强化
4. **词汇专项** - 平日词汇积累

### 建议学习节奏

#### 周末计划（2小时）
1. **完整模考**（90分钟）
   - 选择1套改革后真题
   - 完整完成所有部分
2. **错题记录**（20分钟）
   - 立即记录所有错题到 `errors/` 目录
   - 使用模板格式记录
3. **休息**（10分钟）

#### 平日计划（30-60分钟）
- **选项A**：TPO单项练习（30-40分钟）
  - 选择1个专项（听力/阅读/口语/写作）
  - 完成1-2套练习
  - 记录错题（10分钟）

- **选项B**：词汇专项（20-30分钟）
  - 完成词汇练习
  - 记录不熟悉的词汇

- **选项C**：听力专项（40-60分钟）
  - 完成听力练习
  - 记录错题和听不懂的部分

## 📝 手动使用方式（可选）

如果你想自己管理文件，可以参考以下方式：

### 1. 记录错题

#### 方式一：使用模板文件
复制 `templates/error_template.md` 到对应的 `errors/` 子目录，填写错题信息。

#### 方式二：直接在对应目录创建文件
在 `errors/reading/`、`errors/listening/` 等目录下创建新的markdown文件。

#### 错题记录格式
```markdown
# 错题记录 - [日期]

## 基本信息
- **来源**：TPO XX / 改革后真题 XX
- **日期**：YYYY-MM-DD
- **题型**：阅读/听力/口语/写作
- **题目编号**：第X题

## 题目内容
[粘贴题目内容]

## 我的答案
[你的答案]

## 正确答案
[正确答案]

## 错误原因分析
[简要说明为什么错了]

## 需要AI分析的问题
[列出你想让我帮你分析的具体问题]
```

### 2. 请求AI分析

在错题文件中添加"需要AI分析的问题"部分，然后：
1. 打开错题文件
2. 告诉我："请帮我分析这个错题"
3. 我会读取文件并提供详细分析

### 3. 更新进度

在 `progress/weekly_progress.md` 中记录：
- 本周完成的学习内容
- 错题数量统计
- 学习时间统计
- 需要改进的地方

## 📊 进度追踪

## 💡 学习建议

### 针对ADHD的学习技巧
1. **时间管理**
   - 使用番茄工作法：25分钟专注 + 5分钟休息
   - 设置明确的开始和结束时间

2. **错题管理**
   - 当天完成练习后立即记录错题
   - 不要积累太多，及时处理

3. **进度可视化**
   - 定期查看进度文件，看到自己的进步
   - 庆祝小成就，保持动力

4. **专注力管理**
   - 周末2小时模考时，中间可以休息10分钟
   - 平日练习控制在1小时内
